# Voice-Based Concept Understanding Analyzer (VBCUA)

## Project Overview
The Voice-Based Concept Understanding Analyzer (VBCUA) is an AI-powered web application that evaluates how well a user understands and explains a concept through speech.

## Features
- Speech-to-Text using OpenAI Whisper
- Semantic Similarity using Sentence-BERT
- Fluency Analysis
- Concept Understanding Score
- PDF Report Generation
- Streamlit Web Interface

## Technologies Used
- Python
- Streamlit
- OpenAI Whisper
- Sentence-BERT
- Librosa
- ReportLab

## How to Run

1. Clone the repository

```bash
git clone https://github.com/23F21A3110/VBCUA.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```

## Output
- Speech Transcription
- Concept Similarity Score
- Fluency Analysis
- PDF Report
